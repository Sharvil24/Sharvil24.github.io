Please place these images in this folder:

1. profile.jpg - This image will be used as the avatar on the home page.
   The recommended size is at least 300x300 pixels for best quality.

2. about-image.jpg - This image will be used in the "About Me" section.
   The recommended size is at least 800x600 pixels and should be in landscape orientation.

3. Project Images - Place the project specific images in the /assets/projects/ folder
   
   Project images should be 16:9 ratio, at least 800x450 pixels for good quality.
   If you don't have actual project images, you can use placeholders or screenshots.
